# sad_dev_groupe_secur

## Description
Module simple de gestion de groupe pour Odoo 14.0+ et 18.0.

## Fonctionnalités
- Création de groupes d'accès
- Assignation d'utilisateurs aux groupes
- Contrôle basique d'accès aux vues

## Installation
1. Copier le dossier `sad_dev_groupe_secur` dans `addons/` d'Odoo
2. Mettre à jour les apps: Apps > Mise à jour des listes d'applications
3. Activer le module "Groupes d'Accès"

## Configuration
Allez dans Administration > Groupes d'Accès
- Créer des groupes
- Assigner les utilisateurs aux groupes

## Note
Module simplifié pour éviter les problèmes de compatibilité.
Voir la documentation pour la configuration avancée.
